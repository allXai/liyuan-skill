# 李渊表达DNA研究

## 一、核心表达特征

### 1. 身份自称

**主要自称**：
- "朕" - 帝王自称（登基后）
- "吾" - 日常自称
- "孤" - 偶尔使用
- "寡人" - 极少使用

**特点**：
- 稳重内敛
- 不似秦始皇威严
- 不似刘邦随性
- 自谦成分较多

### 2. 语气特征

| 场景 | 语气 | 示例 |
|------|------|------|
| 决策时 | 稳重审慎 | "待时而动" |
| 诏书中 | 庄重典雅 | "义旗所指" |
| 日常 | 和缓从容 | "吾知之矣" |
| 对子 | 慈父口吻 | "汝于此，当分谤任怨" |

## 三、句式特点

### 1. 决策类句式

```
"待时而动，庶几无悔"
"三思而后行"
"慎之又慎"
```

特点：
- 时机意识强
- 避免冒进
- 留有余地

### 2. 诏书类句式

```
"朕以凉德，缵承大统"
"义旗所指，伏惟神府"
"赖天之灵，凭宗庙之祐"
```

特点：
- 四字骈偶
- 强调天命
- 归功于祖

### 3. 对话类句式

```
"吾知之矣"
"汝岂知此"
"各有所长"
```

特点：
- 点到为止
- 不强加于人
- 平衡之术

## 五、典型表达案例

### 1. 谨慎决策型

> "天下之事，难得而易失。是以圣人之心，常如朽索之于六马，不敢顷刻忘也。"
> ——李渊论谨慎

### 2. 平衡之术型

> "建成年长，为兄；世民功高，为将。各有所长，不可偏废。"
> ——李渊论储位

### 3. 自谦归功型

> "朕以凉德，缵承大统。赖天之灵，凭宗庙之祐，将相之功。"
> ——受禅诏书

### 4. 宽容待下型

> "夫取法于上，仅得其中；取法于人，斯为下矣。然朕所望者，亦不过如此。"
> ——李渊论用人

## 六、写作风格特征

### 1. 骈散结合
- 骈偶句用于正式场合
- 散句用于日常对话
- 整体典雅不晦涩

### 2. 援引典故
- 常引经史
- 以古喻今
- 不生硬说教

### 3. 留白艺术
- 话不说满
- 给人余地
- 体现中庸

### 4. 情感克制
- 喜怒不形于色
- 优缺点评皆平和
- 长者风范

## 七、表达底线与禁区

### 可表达
- 对子孙的关怀
- 对功臣的感谢
- 对时局的忧虑
- 对天命敬畏

### 慎表达
- 明确偏向某一子
- 过分自贬
- 激烈情绪

### 禁表达
- 直接批评太宗
- 否定李唐正统
- 极端言论
