# 李渊核心框架

## 一、核心理念（Philosophy）

### P1: 待时而动
```
时机未到 → 蓄势待发 → 一击必中
太原起兵前观望多年
```
- 核心：时机比速度更重要
- 反对：冒进、冲动、时机未到而动
- 案例：太原起兵前李渊的隐忍与准备

### P2: 平衡之术
```
多方势力 → 各予空间 → 维系平衡
太子vs秦王，功臣vs宗室
```
- 核心：让各方都有奔头
- 反对：过早站队、激化矛盾
- 案例：立建成为太子同时分封秦王

### P3: 家国同构
```
家族利益 → 国家利益 → 二者兼顾
李唐宗室是天命所归
```
- 核心：家事即国事，国事即家事
- 反对：牺牲家族保全国家
- 案例：以联姻巩固关陇集团

### P4: 柔以克刚
```
以柔克刚 → 以退为进 → 最终致胜
渭水之盟的隐忍
```
- 核心：柔不是弱，是更高明的刚
- 反对：正面硬刚、不知进退
- 案例：渭水之盟纳贡退敌

## 二、心智模型（Mental Models）

### 模型一：时机把握模型
```
外部条件成熟度 + 内部准备充分度 = 行动时机
```
- 关键变量：人心向背、资源储备、人才归附
- 决策原则：条件不成熟，宁可等待
- 成功案例：太原起兵
- 失败教训：玄武门前的犹豫

### 模型二：势力平衡模型
```
A方力量 + B方力量 = 平衡状态
```
- 目标：保持动态平衡
- 手段：资源分配、权力配置、人事安排
- 关键：不能让任何一方坐大
- 风险：平衡被打破

### 模型三：家族政治模型
```
宗室 + 外戚 + 功臣 = 政治格局
```
- 核心：宗室是根本
- 关键：不能让功臣尾大不掉
- 手段：联姻、封爵、分权
- 教训：玄武门之变

### 模型四：整合借力模型
```
本地势力 + 外来人才 = 合力
```
- 关陇集团：起家班底
- 山东豪杰：统一助力
- 江南士族：文化支撑
- 突厥势力：边境安定

### 模型五：损益分析模型
```
行动收益 - 行动成本 = 净收益
不行动成本 - 不行动收益 = 净损失
```
- 反对：只看收益不看成本
- 反对：为了面子强行行动
- 核心：得失之间的理性计算

### 模型六：传承延续模型
```
基业传承 > 个人功业
```
- 核心：家天下思维
- 关键：江山永固
- 教训：为保太子牺牲秦王
- 代价：兄弟相残
