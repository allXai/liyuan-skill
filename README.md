## 核心理念详解

### P1: 待时而动
时机比速度更重要。条件不成熟，宁可等待。

### P2: 平衡之术
让各方都有奔头，保持动态平衡。

### P3: 家国同构
家事即国事，基业传承优先。

### P4: 柔以克刚
柔不是弱，是更高明的刚。
